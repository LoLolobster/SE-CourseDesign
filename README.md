# 社团管理系统

​	本项目是基于微信小程序开发的社团管理系统。



## 须知

* 每周三为例会时间，讨论以下事项：
  * 本周计划开发页面；	
  * 接口文档；
* 周中为开发时间；
* 次周周二进行以下工作：
  * 前后端交互测试；
  * 根据本周开发内容填写需产品代办列表（即需求报告）；
  * 填写Sprint开发文档；
